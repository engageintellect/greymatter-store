# greymatter-store

## Description:
This is a simple key-value store that uses a file as a backing store. It is written in Python 3.6 and uses the standard library.

## Technologies:
* Python
* HTMX
* Django
* Stripe
* SQLite
* TailwindCSS

## Getting Started:
1. Clone the repo
2. Create a virtual environment
3. Install the requirements
4. Run the server
